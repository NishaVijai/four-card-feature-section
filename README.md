## Description

**Four Card Feature Section** is a **responsive frontend layout project** built using **semantic HTML and SCSS**, based on a **Frontend Mentor challenge**.

The project focuses on creating a visually balanced four-card layout that adapts seamlessly across screen sizes, demonstrating strong fundamentals in **CSS Grid**, **Flexbox**, and **mobile-first responsive design**.
